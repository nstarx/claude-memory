# Claude Memory

VitePress site documenting Claude's self-assessment of training and capabilities.

## Overview

A documentation site built with VitePress that captures and presents Claude AI's introspective analysis of its own training process, capabilities, and limitations.

## Contents

- `docs/` -- VitePress documentation source files
- `package.json` -- Node.js project configuration
- `package-lock.json` -- Dependency lock file

## Usage

```bash
npm install
npm run docs:dev
```

Then open the local development server URL in your browser.
